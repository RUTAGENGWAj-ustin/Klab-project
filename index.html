👤 About Me

I am Rutagengwa Justin, an Information Technology graduate from the University of Rwanda based in Kigali, Rwanda, with a strong focus on backend engineering, system design, and enterprise application development.

I have hands-on experience designing and developing scalable, secure, and high-performance systems using Java (Spring Boot), Python (Django), and Node.js, as well as building modern front-end applications with React. I also develop cross-platform and native mobile applications using Flutter (Dart) and Java.

My database experience includes PostgreSQL, MySQL, and MongoDB, where I design optimized schemas, implement efficient queries, and ensure data integrity, high availability, and security. I am also experienced in RESTful API design, JWT-based authentication, and third-party integrations, including Mobile Money services (MTN & Airtel).

Currently, I work as a Software Engineer at Dakuita Ltd, focusing on Odoo ERP customization and extension. My work includes developing custom modules, extending and modifying existing models, handling XML configurations, and integrating Odoo with external systems. I also contribute to backend architecture, database optimization, and full-stack feature implementation.

Previously, during my internship at KLAB, I worked on front-end development using Figma, HTML, CSS, and React, translating UI/UX designs into responsive and functional interfaces for applications such as e-commerce platforms.

In addition, I have practical experience in Linux system administration, containerization with Docker, and general system support, enabling me to manage deployment environments and improve system reliability.

I apply strong analytical thinking and problem-solving skills to design efficient solutions and debug complex system issues. I am continuously expanding my expertise in system architecture, cybersecurity, and cloud technologies, with a goal of building robust, scalable, and production-ready software systems.
